# WhatsApp
Este projeto é um clone da interface do WhatsApp Web desenvolvido com HTML, CSS e JavaScript, com foco em design moderno e experiência semelhante ao aplicativo original.
🚀 Funcionalidades
Interface semelhante ao WhatsApp Web
Lista de contatos com avatar
Área de chat interativa
Envio de mensagens
Resposta automática simulada
Scroll automático
Exibição de horário das mensagens
🎨 Tecnologias Utilizadas
HTML5
CSS3 (Flexbox + Grid)
JavaScript (Vanilla)
